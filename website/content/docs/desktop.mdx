# Desktop Distribution

AnyCowork is distributed as a native desktop application using Tauri.

## Build Process

To build the application for your current platform:

```bash
npm run tauri build
```

This command:
1. Builds the React frontend (`dist/`)
2. Compiles the Rust backend in release mode
3. Bundles them into a native installer

## Output Formats

Tauri generates platform-specific installers in `src-tauri/target/release/bundle/`:

### Linux
- **Debian**: `.deb` package (Recommended)
- **AppImage**: Portable executable
- **Path**: `bundle/deb/` and `bundle/appimage/`

### macOS
- **Disk Image**: `.dmg`
- **Application**: `.app`
- **Path**: `bundle/dmg/` and `bundle/macos/`

### Windows
- **Installer**: `.exe` (NSIS)
- **MSI**: `.msi`
- **Path**: `bundle/nsis/` and `bundle/msi/`

## Cross Compilation

To build for other platforms, we recommend using GitHub Actions, as cross-compiling Rust + C dependencies (SQLite) can be complex locally.

See our [GitHub Workflow](https://github.com/AnyCowork/AnyCowork/blob/main/.github/workflows/release.yml) for examples.
