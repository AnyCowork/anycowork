# Getting Started

Get up and running with AnyCowork in minutes.

## Prerequisites

- **Rust** (1.77.2+)
- **Node.js** (18+)
- **pnpm** or **npm**

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/AnyCowork/AnyCowork.git
cd anycowork
```

### 2. Install Dependencies

```bash
# Install frontend dependencies
npm install

# (Optional) Install website dependencies if you want to run the docs
cd website && npm install && cd ..
```

### 3. Setup Environment

Create a `.env` file in the root directory:

```bash
cp .env.example .env
```

Edit `.env` and add your API keys:

```env
OPENAI_API_KEY=sk-...
# Optional: Add other keys as needed
```

### 4. Run the Application

Start the development server:

```bash
npm run tauri dev
```

This will:
1. Compile the Rust backend (this may take a few minutes the first time).
2. Start the Vite frontend server.
3. Launch the AnyCowork desktop application window.

## Usage Guide

### First Steps

1. **Create an Agent**: Click the "New Agent" button in the sidebar. Give it a name like "Research Assistant".
2. **Chat**: Start a conversation. Try asking "Can you summarize the README file in this folder?".
3. **Permissions**: When the agent tries to read a file, a permission card will appear. Click "Approve" to proceed.
4. **Tools**: The agent can use tools like `read_file`, `search_web`, and `run_command` (if enabled).

### Architecture Overview

AnyCowork runs entirely on your machine (Local-First).
- **Frontend**: React + Vite (Port 1420)
- **Backend**: Rust + Tauri (IPC)
- **Database**: SQLite (`~/.local/share/com.anycowork.app/anycowork.db`)

## Troubleshooting

- **Build Fails**: Ensure you have the latest Rust version: `rustup update`.
- **Missing Dependencies**: On Linux, you may need system libraries like `libwebkit2gtk-4.0-dev`. See [Tauri Prerequisites](https://tauri.app/v1/guides/getting-started/prerequisites).
