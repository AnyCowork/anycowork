## Installation

### From PyPI

```bash
pip install anycowork
anycowork-desktop
```

### From Source

```bash
git clone https://github.com/anycowork/anycowork.git
cd anycowork
./setup-conda.sh
```

## Core Concepts

### Agents

Agents are the core of AnyCowork. They process messages, use tools, and maintain conversation history.

```python
from anycowork import Agent

agent = Agent(
    provider="anthropic",
    model="claude-sonnet-4",
    tools=[search, calculate]
)

response = await agent.execute("What's 2+2?")
```

### Tools

Tools extend agent capabilities:

```python
from anycowork import Tool

@Tool
def search_web(query: str) -> str:
    """Search the web"""
    return search_engine.query(query)
```

### Federation

Connect multiple AnyCowork instances:

```python
from anycowork.federation import Node

node = Node(name="desktop")
node.connect_to("192.168.1.100:18790")
```
