### Setup Commands
```bash
# Clone and install
git clone https://github.com/anycowork/anycowork.git
cd anycowork/server
pip install -e .

# Configure
mkdir -p ~/.anycowork
cat > ~/.anycowork/.env <<EOF
AI__PROVIDER=anthropic
ANTHROPIC_API_KEY=your_key
EOF

# Run
anycowork --ws ~/.anycowork
```

### Useful Commands
```bash
# Run server
anycowork --ws ~/.anycowork

# Check health
curl http://localhost:8080/health

# View logs
tail -f ~/.anycowork/logs/anycowork.log
```
