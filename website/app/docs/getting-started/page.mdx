# Getting Started

Get up and running with AnyCowork in minutes.

## Installation Options

### Option 1: Install from PyPI (Recommended)

The easiest way to get started:

```bash
# Install
pip install anycowork

# Run as desktop app
anycowork-desktop

# Or run as server
anycowork --ws ~/anycowork-workspace
```

### Option 2: Standalone Executable

Download pre-built executables from [GitHub Releases](https://github.com/AnyCowork/AnyCowork/releases):

- **Windows**: `anycowork-desktop.exe`
- **macOS**: `AnyCowork.app`
- **Linux**: `anycowork-desktop`

### Option 3: Development Setup

For contributors and developers:

#### Prerequisites

- Conda (Miniconda or Anaconda)
- Node.js 18+

#### Setup Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/AnyCowork/AnyCowork.git
   cd anycowork
   ```

2. Initialize the environment:
   ```bash
   ./setup-conda.sh
   ```

3. Configure your API keys:
   - Edit the `.env` file in your workspace (default: `~/anycowork-workspace/.env`)
   - Add your `ANTHROPIC_API_KEY` (or OpenAI/Gemini keys)

4. Run the development server:
   ```bash
   ./server/scripts/start_dev.sh
   ```

## Access Points

Once running, you can access:

- **Frontend**: [http://localhost:3000](http://localhost:3000)
- **Backend API**: [http://localhost:8080](http://localhost:8080)
- **API Docs**: [http://localhost:8080/docs](http://localhost:8080/docs)

## Configuration

### API Keys

AnyCowork supports multiple AI providers. Configure them in your `.env` file:

```bash
# Anthropic (Claude)
ANTHROPIC_API_KEY=your_key_here

# OpenAI (GPT)
OPENAI_API_KEY=your_key_here

# Google (Gemini)
GOOGLE_API_KEY=your_key_here
```

### Workspace

By default, AnyCowork stores data in `~/anycowork-workspace`. You can change this:

```bash
anycowork --ws /path/to/your/workspace
```
