# Features

Discover what makes AnyCowork the perfect platform for building AI assistants.

## Core Features

### Multi-Provider AI Support

Switch between different AI providers without changing your code:

- **Anthropic Claude** - Latest Claude Sonnet models
- **OpenAI GPT** - GPT-4 and GPT-5 ready
- **Google Gemini** - Gemini 2.0 Flash and beyond

```python
# Switch providers with a single config change
agent = Agent(provider="anthropic")  # or "openai", "gemini"
```

### Federation System

Inspired by ROS2, AnyCowork's federation lets you:

- **Connect Multiple Instances** - Link desktop, mobile, and server deployments
- **Distribute Workloads** - Offload heavy tasks to powerful nodes
- **Real-time Sync** - Changes propagate instantly across all nodes
- **Offline Support** - Local-first design works without network

```python
# Publish to federated topic
node.create_publisher("agent-updates")
await pub({"agent_id": "...", "status": "active"})

# Subscribe from any node
node.create_subscriber("agent-updates", handle_update)
```

### Extensible Skills System

Three ways to extend your agents:

1. **Native Skills** - Python-based tools
2. **Anthropic Agent Skills** - Cloud-based capabilities
3. **MCP Servers** - Model Context Protocol integrations

```python
@Skill
def custom_tool(param: str) -> str:
    """Your custom functionality"""
    return process(param)

registry.register_skill("my-tool", custom_tool)
```

## Developer Experience

### Clean API

```python
# Simple, intuitive API
agent = Agent(
    provider="anthropic",
    model="claude-sonnet-4",
    tools=[search, calculate, browse]
)

response = await agent.execute("Research quantum computing")
```
