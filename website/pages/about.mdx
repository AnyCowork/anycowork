# About AnyCowork

AnyCowork is an open-source, local-first AI assistant platform built with Rust and Tauri.

## Mission

To provide developers with a powerful, privacy-focused AI assistant that keeps data local while leveraging the best AI models available.

## Key Features

- **Local-First**: All data stored locally in SQLite
- **Multi-Provider**: Support for OpenAI, Anthropic, Google, and more
- **Agentic Workflows**: Coordinator-Worker architecture for complex tasks
- **MCP Native**: Built on the Model Context Protocol
- **Privacy-Focused**: Your data stays on your machine
